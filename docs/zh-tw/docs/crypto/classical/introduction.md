# 古典密碼簡介

在古典密碼學中，我們主要介紹單表替代密碼，多表替代密碼，以及一些其它比較有意思的密碼。

值得一提的是，在古典密碼學中，設計者主要考慮消息的保密性，使得只有相關密鑰的人才可以解密密文獲得消息的內容，對於消息的完整性和不可否認性則並沒有進行太多的考慮。
